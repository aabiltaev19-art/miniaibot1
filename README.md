# miniAIBot1

MiniAI Bot — это минималистичный универсальный чат-бот, который отвечает на различные вопросы.

## Цель
Создать простого ИИ-бота с универсальными ответами.

## Возможности
- отвечает на разные вопросы
- поддерживает диалог
- минималистичный интерфейс

## Технологии
- HTML
- CSS
- JavaScript
- GitHub

## Результат
Создан простой универсальный чат-бот с элементами искусственного интеллекта.
